# Nexus Cloud Platform Documentation

## Overview

Nexus Cloud is an enterprise-grade cloud infrastructure platform offering compute, storage, networking, and managed services across 18 global regions.

- G1: 2 vCPUs, 8GB RAM - $0.08/hour

- G4: 8 vCPUs, 32GB RAM - $0.32/hour
- G8: 16 vCPUs, 64GB RAM - $0.64/hour

**Compute Optimized (C-Series)**
- C4: 8 vCPUs, 16GB RAM - $0.28/hour
- C8: 16 vCPUs, 32GB RAM - $0.56/hour
